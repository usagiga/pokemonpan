# pokemonpan
甘きパンよ、来たれ

## About

ポケモンパンに付いてくる [デコキャラシール](https://www.daiichipan.co.jp/decochar/) の収録ポケモン情報を定期的に取得し、
登録した Discord サーバーに対して要約を投稿するスクリプトです。

好きなポケモンが含まれた弾では、メンションすることも可能です。
